<h1> Myntra Clone – React + Redux + Bootstrap </h1>

A fully functional Myntra e-commerce clone built using React, Redux, and Bootstrap.
The project focuses on clean UI, component architecture, global state management, and essential e-commerce features.

<h2>Features</h2> 

Product Listing Page – Displays items with images, price, and details.

Product Details Page – Shows selected product information.

Add to Cart Functionality – Uses Redux for cart state.

Cart Page – Quantity update, remove item, total calculation.

Responsive UI – Built using Bootstrap

State Management – Centralized cart and product state using Redux.

Reusable Components – Navbar, product card, footer, etc.

<h2> Tech Stack </h2>

React.js – UI development

Redux – State management

Bootstrap – Styling and layout

JavaScript (ES6+)

<h2>Installation and Setup</h2>
git clone <repo-url>
npm install
npm start (for backend part)
npm run dev in (for frontend part)(different terminal)
